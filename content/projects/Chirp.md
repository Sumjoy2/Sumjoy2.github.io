## Description

>***Chirp!*** A cozy adventure game where you fly around as a mother bird, searching for her lost eggs through a fantastical city to find what was lost and meet a few friends along the way!

Checkout the [Itch Page][ItchPage] if this sounds interesting to you!!


## Stuff I did on this project

> [!INFO]+ Project Timeline
> This project took around 3 months total. Lasting from *early April* to the *end of June*

### Brief

Dialog system. Helped with bird control / player flight. Publishing to itch.io. Add basic graphical options to support lower end computers. Created rings to fly through. Created quest framework. Increased reactivity of UI buttons.
